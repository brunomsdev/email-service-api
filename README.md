# Email Service API

A simple email microservice built with Node.js and Express.

This service was created as part of a college project to understand:
- Microservice architecture
- API communication with a frontend
- Basic authentication via headers
- CORS configuration

## Tech Stack
- Node.js
- Express
- CORS

## Features
- Token-based request validation
- Endpoint to trigger email sending
- Frontend integration for testing

## Notes
This is a learning-focused project and can be extended with:
- Real email providers (Nodemailer, SendGrid)
- Environment variables
- Authentication middleware
